# 🏥 Advanced Healthcare Analytics & Hospital Performance Dashboard

## 📌 Project Overview
The Advanced Healthcare Analytics Dashboard is developed using Microsoft Power BI to analyze hospital operations, patient admission trends, financial performance, and clinical outcomes.
This project transforms healthcare data into meaningful insights that support operational monitoring and management-level decision-making.

## 🎯 Project Objectives
- Monitor overall hospital performance  
- Analyze patient admission trends  
- Evaluate department-level performance  
- Track recovery and mortality rates  
- Measure bed occupancy efficiency  
- Design KPI-driven performance dashboards  

## 📊 Dashboard Structure
### 🔷 Executive Overview Dashboard
Provides high-level KPIs for performance monitoring:
- Total Patients  
- Total Revenue  
- Average Length of Stay  
- Bed Occupancy %  
- Recovery Rate %  
- Mortality Rate %  
- Monthly Admission Trend  
- Department-wise Revenue Analysis  
Conditional formatting is applied to highlight performance indicators.

### 🔷 Patient Admission & Outcome Analysis
Focused on demographic insights and clinical performance:
- Monthly Admission Trend (Line Chart)  
- Admissions by Department (Bar Chart)  
- Gender Distribution (Pie Chart)  
- Age Group Analysis  
- Donut Chart showing Outcome Distribution (Recovered, Improved, Deceased)

## 🛠 Tools & Technologies Used
- Microsoft Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Conditional Formatting  
- Professional Dashboard Design  

## 📈 Key DAX Measures Implemented
- Total Patients  
- Total Revenue  
- Average Length of Stay  
- Bed Occupancy %  
- Recovery Rate %  
- Mortality Rate %  
- Department Revenue  
- Month-over-Month Growth

## 📂 Dataset Information
- 1000 Patient Records  
- 15 Columns including:
  - Admission Date  
  - Discharge Date  
  - Department  
  - Diagnosis  
  - Treatment Type  
  - Bed Number  
  - Billing Details  
  - Outcome
